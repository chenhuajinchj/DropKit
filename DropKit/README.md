# DropKit

macOS 菜单栏应用，提供文件暂存架和剪贴板历史功能。

<p align="center">
  <img src="DropKit/demo.gif" alt="DropKit Demo" width="480">
</p>

## 功能

- **文件暂存架** - 拖拽文件时摇晃鼠标，快速暂存文件
- **剪贴板历史** - 自动记录复制内容，支持文本、图片、文件
- **菜单栏常驻** - 随时访问，不占用 Dock 空间

## 系统要求

- macOS 14.0 (Sonoma) 或更高版本

## 安装

从 [Releases](https://github.com/chenhuajinchj/DropKit/releases) 下载最新版本。
