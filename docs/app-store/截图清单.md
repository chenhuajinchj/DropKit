# DropKit App Store 截图清单

macOS App 截图有**严格尺寸要求**，先看规则再截。

## 尺寸规则（必须严格符合其一）

App Store Connect 只接受以下尺寸（16:10），RGB、无透明通道、PNG 或 JPG：

| 尺寸 | 说明 |
|------|------|
| 1280 × 800 | |
| 1440 × 900 | |
| 2560 × 1600 | **推荐**（Retina，清晰） |
| 2880 × 1800 | |

- 至少 1 张，最多 10 张。建议 **4～5 张**。
- 所有截图最好用**同一尺寸**。

## 截图怎么截（菜单栏 App 技巧）

1. **整窗截图**：`Cmd + Shift + 4` 然后按 `空格`，点要截的窗口/面板 → 截图存到桌面。
2. DropKit 的搁置架、剪贴板面板、设置窗口都是独立窗口/面板，好截；菜单栏下拉菜单点开后也能截。
3. 截完尺寸基本不会刚好等于上面的值，需要**统一缩放/铺到画布**。用下面命令把一张图铺到 2560×1600 白底画布（不变形）：
   ```bash
   # 把 input.png 居中铺到 2560x1600 白底，输出 out.png
   sips -p 1600 2560 --padColor FFFFFF input.png --out out.png
   ```
   或用「预览」App → 工具 → 调整大小/画布，手动调到 2560×1600。

> 提示：也可以把界面摆在一个浅色背景上整屏截，再裁到 2560×1600，视觉更完整。

## 要截的 5 个画面

| # | 画面 | 怎么呈现 | 想突出 |
|---|------|---------|--------|
| 1 | 菜单栏主菜单 | 点菜单栏 DropKit 图标，展开下拉菜单 | 一眼看懂这是个菜单栏工具 |
| 2 | 剪贴板历史面板 + 搜索 | 打开剪贴板面板，里面有几条历史，最好展示搜索框 | 核心卖点：剪贴板历史可搜索 |
| 3 | 搁置架（收起态） | 拖文件时呼出的浮动搁置架，收起小窗 | 拖拽暂存文件 |
| 4 | 搁置架（展开态） | 搁置架展开，网格里有几个文件缩略图 | 文件暂存区全貌 |
| 5 | 设置窗口 | 设置页，展示快捷键 + 监控文件夹配置 | 可定制、原生体验 |

## 截图前的准备

- 用**真实界面**截，别用拼接/示意图（与实际不符会被拒）。
- 面板里放些**真实但不敏感**的示例内容（别露隐私/密码/真实聊天）。
- 浅色或深色模式选一种统一风格。

截好的图最终在 App Store Connect → 版本 → "App 预览和截图" 上传。
